# Smart-Blind-Guide-Cane
A self-developed intelligent guidance system for visually impaired people

---

## 写在前面：一个高中生的两年
大家好，我是蒲阳，一名普通班的高中生。

两年前，我在一次社区志愿活动中，亲眼看到一位视障老人在路口踌躇了十几分钟，最终还是被车流惊得后退了好几步。那一刻我下定决心，要做一套真正能让盲人“走得放心、走得安心”的设备。

从零基础啃C++算法、画PCB板、调试大模型，到自己建模3D打印外壳、反复减重优化佩戴体验，整整两年，我几乎把所有课余时间和生活费都砸在了这个项目上。我没有团队，全程都是一个人和GitHub摸索。

我带着它参加了一场科创比赛，本想拿到一点奖金，继续把它做下去。可最终，那些网购拼装的“玩具作品”拿了奖，而我这套完整自研、真正为视障群体设计的设备，却落榜了。

我很委屈，也很迷茫。我没有钱申请专利，也没有资源继续迭代。思来想去，我决定把它**完全开源**。

---

## 开源的目的
1.  **给视障群体一份真正能用的工具**
    我不想让它烂在我的硬盘里。哪怕只有一个盲人能因为这个项目走得更安全，我这两年的付出就不算白费。
2.  **守住首创权，也守住善意**
    我将自己独立完成的硬件设计、场景适配、功能优化部分全部公开。GitHub的提交记录与时间戳，会永久证明，我是这套「轻量化智能助盲系统」完整方案的独立原创者。
3.  **抛砖引玉，希望有人能把它做得更好**
    我只是个高中生，能力有限。如果你是工程师、开发者、公益从业者，欢迎你fork、修改、优化它，让它变得更稳定、更轻便、更实用。

---

## 项目说明
本项目是一套集成了**主动避障、语音交互、震动提醒、人体工学轻量化设计**的智能助盲设备，包含：
- 嵌入式控制代码：基于C++开发，核心框架参考虾哥教程，避障逻辑、交互流程、硬件适配均为独立实现。
- 原创硬件设计：包含完整的PCB原理图、工程文件与优化后的3D打印外壳模型。
- 轻量化AI交互：参考虾哥的大模型部署思路，针对助盲场景做了精简与适配，可实现本地离线交互。
- 完整设计文档：包含项目调试日志、优化记录与功能说明。
